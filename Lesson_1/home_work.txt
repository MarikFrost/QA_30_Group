1) Посмотреть где я:
 - pwd
2) Создать папку:
	- mkdir folder_1
3) Зайти папку:
	- cd folder_1
4) Создать три папки:
	- mkdir folder_2 folder_3 folder_4
5) Зайти в любую папку:
	- cd folder_2
6) Создать пять файлов (три txt, два json)
	- touch file_1.txt file_2.txt file_3.txt file_4.json file_5.json
7) Создать три папки:
	- mkdir folder_5 folder_6 folder_7
8) Вывести список содержимого папки:
	- ls
	- ls -la
9) Открыть любой txt файл:
	- cat file_1.txt  (открыть в командной строке)
	- vim file_1.txt     (открыть в редакторе VIM)
10) Написать туда что-нибудь, любой текст:
	- cat >> file_1.txt (добавляем строку в файл)
	- vim file_1.txt    (добавить строку можно также через редактор VIM нажимаем i для редактирования)
11) Сохранить и выйти:
	- Eсли добавляли данные через cat, тогда жмем ctr+c ;
	- Через редактор VIM, жмем :wq ;
12) Выйти из папки на уровень выше:
	- cd ..
13) Переместить любые 2 файла, которые вы создали, в любую другую папку:
	- mv folder_2/file_1.txt folder_2/folder_5/file_1.txt
	- mv folder_2/file_4.json folder_2/folder_5/file_4.json
14) Скопировать любые 2 файла, которые вы создали, в любую другую папку:
	- cp folder_2/file_2.txt file_2.txt
	- cp folder_2/file_5.json file_5.json
15) Найти файл по имени:
	- find . -name "file_3.txt"
16) Посмотреть содержимое в реальном времени:
	- tail -F file_2.txt
17) Вывести несколько (неизвестно сколько строк нужно выводить) первых строк из текстового файла
	- head -3 file_2.txt
18) Вывести несколько (неизвестно сколько строк нужно выводить) последних строк из текстового файла
	- tail -3 file_2.txt
19) Посмотреть содержимое длинного файла:
	- less file_2.txt
20) Вывести дату и время:
	- date +"Сегодня: %d %B %Y %R"
21) Отправить HTTP запрос на сервер: 
	- curl "http://162.55.220.72:5005/get_method?name=Dmytro&age=31"
